# nth_prime
What is the 10,001st prime number?

# Install & Use
First clone the repo:
```
git clone https://github.com/AlexLandherr/nth_prime.git
```
Change directory to repo:
```
cd nth_prime/
```
Create a directory called `obj`:
```
mkdir obj
```
Then build & compile using `make`:
```
make
```

Run:
```
./nth_prime
```
